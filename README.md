# Калькулятор

Браузерный калькулятор с современным дизайном.

## Возможности

- Базовые математические операции (сложение, вычитание, умножение, деление)
- Поддержка десятичных чисел
- Удаление последнего символа
- Очистка дисплея
- Поддержка клавиатуры

## Локальный запуск

```powershell
pnpm install
pnpm dev
```

## Деплой на Vercel

### Способ 1: Через Vercel CLI

1. Установите Vercel CLI (если не установлен):
```powershell
npm install -g vercel
```

2. Задеплойте проект:
```powershell
vercel
```

3. Для продакшн-деплоя:
```powershell
vercel --prod
```

### Способ 2: Через GitHub

1. Загрузите проект в репозиторий GitHub
2. Перейдите на [vercel.com](https://vercel.com)
3. Войдите в аккаунт и нажмите "New Project"
4. Импортируйте ваш репозиторий
5. Vercel автоматически определит настройки и задеплоит проект

### Способ 3: Через веб-интерфейс Vercel

1. Перейдите на [vercel.com](https://vercel.com)
2. Нажмите "New Project"
3. Загрузите папку проекта или перетащите файлы
4. Нажмите "Deploy"
